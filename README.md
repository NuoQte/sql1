# MrDB

"An asynchronous Python ORM that supports MySQL, SQLite, and PostgreSQL with robust performance and easy-to-use interface."  
  
**mrdb** supports the modules **aiomysql**, **aiosqlite**, and **asyncpg**.



### Install
```bash
pip3 install -U mrdb
```
